Задача:
	Написать веб-сервис, который реализует следующие методы:

		String getCreate(String schemaName, String tableName) – возвращает DDL на создание указанной существующей в БД таблицы. Скрипт должен содержать DDL создания первичного ключа.
		String getSelect(String schemaName, String tableName) – возвращает запрос вида SELECT <список полей через запятую> FROM <указанная таблица> WHERE <поле-первичный ключ> = ? (либо писок таких полей в случае составного ПК)
		String getUpdate(String schemaName, String tableName) – возвращает запрос вида UPDATE <указанная таблица> SET <поле1>=?, <поле2>=?..... WHERE <поле-первичный ключ >=? (либо писок таких полей в случае составного ПК))

	Считается, что у пользователя СУБД имеются необходимые права.

	Результат выполнения задания: war файл) для разворачивания на tomcat + инструкция по настройке подключения к БД + исходный код.
	
Поскольку в формулировке задания отсутствует указание на конкретную БД, на усмотрение разработчика была выбрана база MySQL. 
Архитектура приложения позволяет достаточно легко добавлять поддержку других баз данных.
	
Содержимое каталога:
1. mysql-connector-java-5.1.40-bin.jar - драйвер для БД MySQL
2. readme.txt - даная инструкция
3. SQLWebService_war.war - веб-архив для разворачивания на tomcat (в процессе разраобтки использовалась версия 8.0.17)
4. SQLWebService.zip - архив проекта для IDE Intellij IDEA (В процессе разработки использовалась версия Ultimate 2016.2.4)

Настройка подключения tomcat к БД.

	Для настройки подключения к БД требуется отредактировать файл context.xml, находящийся в каталоге %CATALINA_HOME%/conf (CATALINA_HOME - каталог установки tomcat)
	В файл внутри тэга <context> необходимо добавить следующий фрагмент:
	<Resource name="jdbc/TestDB" auth="Container" type="javax.sql.DataSource"
               maxActive="100" maxIdle="30" maxWait="10000"
               username="username" password="password" driverClassName="com.mysql.jdbc.Driver"
               url="jdbc:mysql://localhost:3306/test"/>
	В приведенном фрагменте необходимо изменить аттрибуты "username", "password" и "url" согласно настройкам стенда тестирования.
	
	Файл  SQLWebService_war.war необходимо скопировать в каталог %CATALINA_HOME%/webapps
	Файл mysql-connector-java-5.1.40-bin.jar необходимо скопировать в каталог %CATALINA_HOME%/lib
	Предполагаем, что все необходимые javaee-библиотеки уже присуствуют в каталоге %CATALINA_HOME%/lib
	
	После запуска сервера, на странице http://<tomcat_host>:<tomcat_port>/SQLWebService_war/dbservice будет доступна информация о реализованном web-сервисе. 
	В том числе - его WSDL, используя который, можно посылать запросы сервису(например, при помощи утилиы SoapUI)
